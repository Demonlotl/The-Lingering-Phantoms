The Lingering Phantoms

Description

The Lingering Phantoms, a game inspired by the Backrooms. This game is about a skyscraper that has 522 floors. You'll start from the top floor travelling to the bottom. When you get to floor 521 and below, you will start encountering... phantoms. Each floor's phantom differs from each other, sometimes one can be a combination of all sorts. It is also a multiplayer game so you can join your friends and have fun. The game is also a 3-dimensional game, so you can have a 360 view of what your surrounding is.
